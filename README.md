# 포포팝! (PawPaw Pop!)

포포팝은 모모·루루·보미·두두·피요·누누·토리·콩이, 여덟 동물 친구를 맞추는 오리지널 브라우저 매치3 게임입니다. 모바일 터치와 PC 마우스·키보드에서 모두 플레이할 수 있습니다.

## 주요 기능

- 초기 매치가 없고 항상 유효한 이동이 존재하는 시드 기반 8×8 보드
- 가로·세로 3개 이상 매치, 복수 매치, 캐스케이드, 자동 셔플
- 가로/세로 로켓, 포포 폭탄, 무지개 별과 모든 특수 블록 조합
- 코드로 직접 그린 8종 캐릭터, 고유 실루엣·표정·색각 보조 문양
- 나무 상자, 젤리, 돌, 별 조각 장애물과 복합 목표
- 이동/시간 제한, 점수·콤보·별·보너스·결과 카운트업
- EASY·MEDIUM·HARD·VERY HARD 난이도와 매 판 달라지는 퍼즐·목표·장애물
- 날짜 시드 기반 60초 오늘의 도전과 8종 업적
- 드래그, 두 번 탭/클릭, 방향키, Enter/Space, Escape 조작
- 셔플·망치·포포 아이템, 일시정지와 3-2-1 재개
- Web Audio API로 합성한 효과음/배경 루프, 진동 피드백
- 음소거, 진동, 화면 흔들림, 파티클, 모션 감소 설정
- 손상된 저장 데이터 자동 복구와 `localStorage` 진행 저장
- 360×640부터 1920×1080까지 대응하는 반응형 캔버스

그래픽과 효과음은 외부 이미지·효과음 없이 Phaser Graphics와 Web Audio API로 코드에서 생성합니다. 배경 음악은 사용자가 제공한 `기본 bgm.wav`를 게임용 루프 자산으로 편입해 메뉴·일반 게임·오늘의 도전 상황에 맞게 재생 속도와 음량을 조정합니다.

## 바로 플레이

[휴대폰과 PC에서 포포팝 실행하기](https://rwajeo.github.io/pawpaw-pop/)

## 실행

Node.js 20 이상을 권장합니다.

```bash
npm install
npm run dev
```

개발 서버 주소는 기본적으로 `http://127.0.0.1:5187`입니다.

## 빌드와 테스트

```bash
npm test
npm run build
```

프로덕션 결과물은 `dist/`에 생성됩니다.

## 조작 방법

- 마우스/터치: 캐릭터를 인접 방향으로 드래그
- 마우스/터치: 첫 캐릭터를 누른 뒤 인접 캐릭터를 눌러 교환
- 키보드: 방향키로 포커스 이동, Enter/Space로 선택
- Escape: 선택 취소 또는 일시정지
- 아이템: 하단 아이템 버튼을 누른 뒤 보드의 대상을 선택

## 프로젝트 구조

```text
src/
  main.ts
  game/
    board/       순수 TypeScript 보드·매치·이동·특수 블록 엔진
    data/        캐릭터, 난이도별 동적 퍼즐, 업적 데이터
    entities/    코드 드로잉 캐릭터, 타일, 파티클/텍스트 풀
    scenes/      부트, 메뉴, 난이도 선택, 게임, 결과, 설정 장면
    systems/     저장, 점수, 콤보, 목표, 오디오, 진동, 업적
    ui/          버튼, 모달, 진행바, 토스트, 튜토리얼
  styles/        반응형·safe-area·접근성 CSS
tests/           보드 엔진과 시스템 단위 테스트
```

## 브라우저 지원

최신 Chrome, Edge, Safari, Firefox를 대상으로 합니다. 진동은 `navigator.vibrate`를 지원하는 브라우저에서만 동작하며, 미지원 환경에서는 자동으로 건너뜁니다. 오디오는 브라우저 자동재생 정책에 따라 첫 사용자 입력 이후 시작합니다.

## 저장 데이터 초기화

게임의 `설정 → 저장 데이터 초기화`를 사용합니다. 개발자 도구에서 직접 지우려면 사이트 저장소의 `pawpaw-pop-save-v1` 항목을 삭제할 수 있습니다.
